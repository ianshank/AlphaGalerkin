## Questions vs. bugs

| I want to… | Use |
| --- | --- |
| Ask a usage question / discuss an idea | [GitHub Discussions](https://github.com/ianshank/AlphaGalerkin/discussions) (or an issue tagged `question` if Discussions is disabled) |
| Report a reproducible bug | [Open a bug report](https://github.com/ianshank/AlphaGalerkin/issues/new?template=bug_report.md) |
| Request a feature | [Open a feature request](https://github.com/ianshank/AlphaGalerkin/issues/new?template=feature_request.md) |
| Report a security vulnerability | See [SECURITY.md](SECURITY.md) — **do not** open a public issue |
| Contribute code or docs | See [CONTRIBUTING.md](CONTRIBUTING.md) |

## When filing an issue

Include your OS, Python version, how you installed (`pip install -e ".[dev]"`?),
the exact command, and the full traceback. A minimal reproduction gets a faster
answer than a description.

This is a research project maintained on a best-effort basis — please be patient
and kind. See our [Code of Conduct](CODE_OF_CONDUCT.md).
